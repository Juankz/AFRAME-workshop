# AFRAME workshop

## Requirements
 - Node 12

## Setup

Execute `npm install`

## Run

- Server: `npm start`
- Aframe watcher: `npm watch`

## Topics

- Introduction to AFRAME, Entities and AFRAME watcher
- Add 3D models
- Add camera
- Add a sky
- Add lights: Illumination basics and types of light
- Components. Modify a model material
- Adding video
- Custom loading
